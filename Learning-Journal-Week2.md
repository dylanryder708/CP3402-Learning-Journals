### Content Insights
- I find that static websites are simple but very limited, because every update requires me to manually edit files, which quickly becomes slow, repetitive, and hard to maintain.
- When I work with dynamic websites, I appreciate the flexibility of database-driven content, but I also notice they introduce new challenges such as extra server load, security concerns, and more complex troubleshooting.
- Using CMS platforms makes sense to me because they save time and avoid reinventing the wheel, providing built-in tools for content management and user interaction, even though they require more server resources and setup.

### Career/Employability/Learning Insights
- Spent some time this week setting up Vagrant and VirtualBox on my computers. I did this ahead of time to avoid any issues the upcoming weeks and make sure I was prepared ahead of time before running into issues closer towards due dates.
- One thing I found out this week was Joomla 6.0.3 doesn't seem to work on scotch-box 3.5 because Joomla 6.0.3 requires a newer version of PHP to run. I did manage to find different LAMP stack called Cognac Box (https://github.com/reddingwebpro/cognacbox) which has similar but newer features and allows Joomla 6.0.3 to run on it. This is great for me because I can use everything I learnt this week through this prac for Assignment_01 without having to use out of date software which could create issues.
- This also allowed me to gain some additional experience with stuff I am unfamiliar with and would not have gotten that experience if I just decided to install an older version of Joomla on my localhost without persisting to get the newest version to work.
